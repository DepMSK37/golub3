## Деплой на Render.com (бесплатно)

### 1. Загрузить на GitHub
```bash
git init
git add .
git commit -m "Голубь v5.0"
git remote add origin https://github.com/ВАШ_АККАУНТ/golub.git
git push -u origin main
```

### 2. Создать Web Service на Render.com
- **Runtime:** Node
- **Build Command:** `npm install`
- **Start Command:** `npm start`
- **Instance Type:** Free

### 3. Добавить Persistent Disk (ОБЯЗАТЕЛЬНО!)
- Disk Name: `golub-data`
- Mount Path: `/data`
- Size: 1 GB (бесплатно)

### 4. Переменные окружения
```
DB_PATH=/data/golub.db
PORT=3000
```

### @karnizcal — Калькулятор карнизов
Диалоговый бот. Пишите числами:
- **1** — Прямые карнизы
- **2** — Г-образные карнизы
- Затем выбираете режим (1/2/3) и вводите размеры в см
- /menu — главное меню, /new — новый расчёт

## Группы
- Создатель группы = администратор
- Администратор может:
  - Менять название группы (карандаш в шапке или в инфо группы)
  - Менять аватарку группы (клик по аватарке в инфо группы)
  - Добавлять и удалять участников
- Все участники видят список членов группы

## Онлайн-статус
- Зелёная точка в списке чатов = человек онлайн
- «в сети» в шапке личного чата
- Обновляется в реальном времени через WebSocket
